# Open Graph Image Required

Please add an Open Graph image with the following specifications:

- **Filename:** og-image.jpg
- **Location:** /public/og-image.jpg
- **Dimensions:** 1200x630 pixels
- **Format:** JPG or PNG
- **Content:** Logo, site name "Çarpım Tablosu", and tagline

This image is referenced in:
- app/layout.tsx (OpenGraph metadata)
- app/page.tsx (Schema.org structured data)

Without this image, social media shares will not display a preview image.
